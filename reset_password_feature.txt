User Story:
As a registered user,
I want to reset my password if I forget it,
So that I can regain access to my account securely.

Acceptance Criteria:
1. The user can request a password reset by providing their registered email address.
2. The system sends a password reset link to the userâ€™s email.
3. The reset link expires after a defined period (e.g., 1 hour).
4. When the user clicks the reset link, they are redirected to a "Reset Password" page.
5. The user must enter a new password and confirm it.
6. The new password must meet the password policy (e.g., minimum length, must include numbers/special characters).
7. After successful reset, the user can log in using the new password.
8. If the reset link is invalid or expired, the user sees an appropriate error message.
